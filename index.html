Array

-> In js array is a collection of elements with different data type.(except c and c++)
-> In c and c++ array is a collection of similar datatype.

PUSH AND POP CONCEPT IN (these are inbuild functions.)

-> push() : Insert from end.
-> pop() : Remove from end.

SHIFT AND UNSHIFT OPERATIONS

-> unshift() : Inserting the elments in the starting of the array a[0].
-> shift() : Removing the elments in the starting of the array a[0].

MORE METHODS

-> indexOf() : It finds the index of the element. and if we try to find the index of element which is not present in the array then it gives -1 index value.
-> splice(index, no. of elements to delete, elements to insert)
-> includes() : It checks the element entered is present in the array or not, if present then it returns true else returns false value.
